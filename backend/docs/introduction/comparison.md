> 在投标 AI 这个赛道，我们不是第一个。
> 下面是 BidWriter 与主要竞品 / 参考项目的对比。

## 详细对比

### 1. 桌面 AI 工具（yibiao / 易标）

| 维度 | yibiao | BidWriter |
|---|---|---|
| **形态** | Electron 桌面单机 | Web SaaS + 桌面备 |
| **用户** | 个人投标员 | 中小团队 |
| **AI 模型** | 用户自配单模型 | 多模型路由自动选 |
| **协作** | 无 | 实时多角色协作 |
| **数据** | 本地 SQLite | 云端 PG + pgvector |
| **计费** | 免费开源 | 按订阅 + Token |
| **知识库** | 本地目录树 | 云端三层知识库 |
| **部署** | 用户本地 | 云端 + 私有化 |
| **版本** | v2.15.1（成熟） | v0.1（M1 起步） |

**BidWriter 优势**：
- 团队协作（v0.1 规划中）
- 多模型路由（yibiao 没有）
- 云端知识库（跨设备）

**yibiao 优势**：
- 完全开源免费
- 已经生产验证（v2.15.1，6 万行代码）
- 桌面离线可用

**借鉴关系**：
- 工作流抽象（Step01-05）
- 任务状态机（paused / restoring / auditing）
- 全文一致性审计
- Prompt 缓存策略
- JSON 修复链路

### 2. SaaS 投标平台（投标易 / 蜜蜂 AI 标书）

| 维度 | 投标易类 | BidWriter |
|---|---|---|
| **核心** | 标书管理系统 + AI 辅助 | AI 自动撰写 + 协作 |
| **AI 能力** | 较弱（句子润色） | 强（多步生成 + 审计） |
| **差异化** | 项目管理 + 流程 | AI 写作质量 + 路由 |
| **数据** | 自己的格式 | Markdown + 标准 docx |
| **导出** | 平台格式 | 通用 .docx |

**BidWriter 优势**：
- AI 能力深度（多步推理、agent 审计）
- 模型路由（成本/质量自动优化）
- 不绑定格式（输出标准 .docx）

**投标易类优势**：
- 完整的投标管理流程（已与采购平台对接）
- 客户关系管理
- 多年行业数据积累

### 3. 国际产品（RFPIO / Loopio / AutogenAI）

| 维度 | RFPIO 类 | BidWriter |
|---|---|---|
| **市场** | 全球（英语为主） | 中国（中文优先） |
| **法规** | SOX / GDPR 合规 | 国密 / 等保合规 |
| **模板** | 通用 RFP 库 | 6 个国内行业模板 |
| **AI 能力** | LLM 微调 | 多模型路由 + 行业适配 |
| **价格** | $$$（企业级） | ¥（中小企业友好） |

**BidWriter 优势**：
- 中文场景深度优化（行业术语、政策合规）
- 价格友好（中小企业付得起）
- 私有化方案（国内合规要求）

**RFPIO 类优势**：
- 国际化（多语言）
- 客户案例库（10+ 年积累）
- 企业级安全认证

## 我们的独特位置

```
              AI 能力深度
                  ▲
                  │
    AutogenAI ●   │
                  │
                  │         ● BidWriter
                  │              ← 在 AI 深度 和 中小市场 之间找平衡
                  │
    投标易 ●       │
                  │
                  │  ● yibiao
                  │
                  └──────────────────────► 团队协作能力
```

**BidWriter 的独特定位**：
1. **AI 深度** — 不止润色，要"自动撰写 + 审计"
2. **多模型路由** — 成本/质量自适应，企业省钱
3. **团队协作** — 不止单人，是团队工作流
4. **中文优先** — 行业术语、政策合规深度优化
5. **价格友好** — 中小企业付得起（¥499/月起）

## 何时选谁

| 需求 | 推荐 |
|---|---|
| 个人投标、零成本 | yibiao（开源） |
| 5-50 人团队、效率优先 | **BidWriter** |
| 大企业、全流程管理 | 投标易类 |
| 跨国业务、英文 | RFPIO / AutogenAI |
| 政府/央企、强合规 | BidWriter Enterprise（私有化） |
